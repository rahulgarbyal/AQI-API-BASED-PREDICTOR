AQI API BASED PREDICTOR 

The Air Quality Index (AQI) Prediction & Monitoring System is a software application that collects real-time air quality data from online APIs, 
processes it, stores it in a database, and predicts AQI levels for locations where data may not be available.

The system uses APIs such as World Air Quality Index Project and OpenAQ to fetch environmental data like AQI, 
temperature, humidity, and pollutant levels. This data is then cleaned and stored using a database like MongoDB.

The project also aims to use machine learning techniques to predict AQI for cities without monitoring stations, improving accessibility of air quality information.
